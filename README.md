**Python Socket Programming & Port Scanner**
Simple Client-Server Communication and Port Scanning using Python Sockets

**Project Description**
This project demonstrates basic **TCP socket programming** using Python.  
It includes:

- A TCP Server
- A TCP Client
- A Simple Port Scanner

The project shows how network communication works using sockets and how open ports can be detected on a target host.

**Features**

TCP Client-Server communication  
Message exchange between client and server  
Localhost connection testing  
Basic port scanning functionality  
Timeout handling for scanning  

**Technologies Used**

- Python 3
- Built-in `socket` library
